# Video Player

Custom video player built with Three.js that composites a background image and video on a canvas, with a synced transcript sidebar.

## Deployed Link
You can visit the deployed version - trupeer-assignment.netlify.app

## Tech Stack

- React 19 + TypeScript
- Three.js via @react-three/fiber + @react-three/drei
- Zustand for state management
- Tailwind CSS v4
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build    # outputs to dist/
npm run preview  # preview production build locally
```

Configured for Netlify deployment via `netlify.toml`.
